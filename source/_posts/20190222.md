css3如何实现边框阴影

```
div
{
box-shadow: 10px 10px 5px #888888;
}
```
![Alt text](/1.png)

语法
```
box-shadow: h-shadow v-shadow blur spread color inset;
```
注释：box-shadow 向框添加一个或多个阴影。该属性是由逗号分隔的阴影列表，每个阴影由 2-4 个长度值、可选的颜色值以及可选的 inset 关键词来规定。省略长度的值是 0。
